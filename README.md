# Modelagem Banco de Dados - Sistema de Gestão Comercial

Este repositório contém os diagramas e scripts SQL do banco de dados do Sistema de Gestão Comercial.

## Conteúdo

- `/diagrams` - Diagramas ER e outros diagramas de modelagem
- `/sql` - Scripts de criação e dados de exemplo

## Como usar

1. Abra os diagramas na ferramenta correspondente (Draw.io, Lucidchart, etc.).
2. Se quiser testar o banco, execute os scripts SQL no seu SGBD.
